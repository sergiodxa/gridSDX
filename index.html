Documentación

El sistema de grid cuentas con las siguientes clases:

  • .contenedor: esta clase sirve para definir el contenedor del sitio y limitar de esta forma el ancho máximo y centrarlo.
  • .fila: esta clase indica cual elemento es una fila que contiene columnas.
  • .col-X: esta clase que varía su valor en X dependiendo del ancho de la columna indica de que tamaño eun elemento.

Para poder utilizar el sistema lo primero es definir el contenedor del sitio (si no queremos que ocupe siempre el 100% del ancho), para hacerlo le agregamos a un elemento la clase .contenedor que centra al elemento, le da un ancho de 99% y un ancho máximo de 960px o 1200px dependiendo del ancho.

El siguiente paso es crear una fila, para esto le agregamos la clase .fila a un elemento, esto le quita al elemento el tamaño de fuente (para solucionar el problema del espacio entre columnas al usar inline-block) y oculta el contenido que sobresalga de la fila.

El último paso es colocar las columnas, para eso dentro de nuestra fila creamos varios elementos con clases .col-X donde en X colocamos el ancho de la columna a crear, todas las clases .col-X le vuelven a colocar el tamaño de fuente en base al del body, cambian el tipo de display a inline-block y alinea verticalmente la columna hacia arriba.

Los siguientes son los anchos posibles para las columnas, si necesitas utilizar más de 5 columnas podes combinarlas agregando la clase .fila a un elemento con una clase .col-X y dentro colocar más columnas.
